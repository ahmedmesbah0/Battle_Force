# Quick Start - 10 Seconds

## Run the Game NOW

```bash
./run.sh
```

**That's it!** The game will launch immediately.

---

## First Time Setup (if needed)

```bash
./setup.sh
```

Then run:
```bash
./run.sh
```

---

## Controls

- **WASD** - Move
- **Mouse** - Aim
- **Left Click** - Shoot
- **R** - Reload
- **1-4** - Switch weapons
- **ENTER** - Start game
